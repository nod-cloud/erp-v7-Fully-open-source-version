#Environmental requirements
1. PHP version 7.3+
MYSQL version 5.6+
3. MYSQL needs to disable strict mode
4. Please use the Google Chrome kernel for the terminal

#Program installation
1. Copy all files under src/to the root directory of the site.
2. After accessing your site address, the installation wizard will be automatically executed.
3. Pseudo static information needs to be configured, APACHE is already built-in, and IIS and NGINX need to be configured separately.
If you encounter any other problems, please contact customer service for assistance.


#Front end engineering files
1. Install and configure node and npm correctly
2. Enter the CRM directory on the command line and execute npm install
3. Execute npm run serve from the command line

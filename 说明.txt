点可云进销存系统

# 关于产品
点可云进销存系统是一款由点可云软件中心出品的高质量付费产品。

# 目录说明：
src/ 程序产品
project/ 前端工程文件
about/ 帮助目录

# 环境要求
1.PHP版本7.3+
2.MYSQL版本5.6+
3.MYSQL需关闭严格模式
4.终端请使用谷歌浏览器内核

# 程序安装
1.将src/下的文件全部拷贝到站点根目录。
2.访问您的站点地址后将自动执行安装向导。
3.需要配置伪静态信息，APACHE已经内置,IIS和NGINX需单独配置。
4.如遇到其他问题请联系客服获取帮助。


# 前端工程文件
1.安装并正确配置node和npm
2.命令行进入crm目录执行npm install
3.命令行执行npm run serve
